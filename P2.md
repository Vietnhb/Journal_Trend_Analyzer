
**Hùng - Project Setup + Core Architecture**
- Tạo project Flutter.
- Cấu trúc thư mục:
  - `lib/core/`
  - `lib/data/`
  - `lib/presentation/`
- Cấu hình `pubspec.yaml`: `http`, `provider`, `fl_chart`.
- Làm theme chung, constants, formatter, error classes.
- Tạo `main.dart`, navigation chính 3 tab: Search, Trends, Dashboard.

**Minh - OpenAlex API + Data Layer**
- Làm model `Publication`.
- Parse dữ liệu từ OpenAlex:
  - title
  - year
  - citation count
  - journal/source
  - authors
  - DOI
  - abstract
- Làm `OpenAlexApiService`.
- Làm `PublicationRepository`.
- Xử lý lỗi API, timeout, no internet, invalid JSON.
- Đảm bảo không có backend, app gọi OpenAlex trực tiếp.

**Phúc - Search Screen + Publication Detail**
- Làm màn hình Search.
- Form nhập topic.
- Hiển thị danh sách publication.
- Loading / empty / error UI cho search.
- Làm card publication.
- Làm màn Publication Detail:
  - title
  - authors
  - year
  - journal
  - citations
  - DOI
  - abstract

**Bảo - Trend Analysis + Chart**
- Xử lý logic group publication theo năm.
- Làm màn Trend Analysis.
- Vẽ chart bằng `fl_chart`.
- Hiển thị:
  - publications by year
  - most active year
  - ranked list các năm có nhiều bài nhất
- Đảm bảo chart responsive trên mobile.

**Việt - Dashboard + Ranking Analytics + Integration Polish**
- Làm Dashboard tổng hợp:
  - total publications
  - average citations
  - most active year
  - top journal
  - top author
  - most influential paper
- Làm ranked list:
  - top influential papers
  - top research journals
  - top contributing authors
- Kết nối UI với provider/state chung.
- Kiểm tra flow toàn app, sửa lỗi giao diện, responsive Android.

Phân chia theo module sẽ gọn như này:

| Người | Phần chính | Folder/File chính |
|---|---|---|
| Hùng | Setup + architecture | `main.dart`, `core/` |
